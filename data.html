Data Available for Download


The data available here are the various data sets I have collected and cleaned. Most of these are related to crime, though some are merely government data or data I am interested in. Since I have cleaned all this data myself, if you find any bugs please email me. You should also email me if you have any requests or suggestions for data.

The data set that are used to create the tools for this site are far more comprehensive than what is available through the tools (i.e. all variables for all agencies in all years are available to download). For a full description of what is available in each data set, click the link to go to the download page on openICPSR. On that page I include a description of the data and precisely what I did to "clean" it.

These data are freely available to download and use. If you do use these data, please cite.
